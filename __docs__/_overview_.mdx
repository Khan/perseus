# Perseus Documentation

Perseus is Khan Academy's exercise question editor and renderer. This Storybook provides documentation and interactive examples of the various components, renderers, and widgets available in Perseus.

## Navigation Structure

- **Overview** - Introduction to Perseus
- **Widget Gallery** - Overview of all available widgets
- **Theming** - Theming options for Perseus components
- **Perseus** - Detailed documentation
  - **Components** - Reusable UI components
  - **Renderers** - Content rendering components
  - **Widgets** - Interactive widgets
    - Overview - General widget information
    - A11Y - Accessibility considerations
    - Widget Demo - Interactive demos
    - Editor Demo - Widget editing interfaces
    - Widget States Gallery - Visual states
    - Widget Internal Components - Internal structure
    - Regression Test - Test cases

*Note: This is a proof of concept showing only the RadioNew widget implementation.*
