# Perseus Editors

This section contains the editor components used in Perseus for creating and modifying content. The editors provide interfaces for content creators to build and configure interactive educational content.

## Available Editors

Perseus includes several editor components:

- **Item Editor** - For creating and editing exercise items
- **Article Editor** - For creating and editing articles
- **Content Preview** - For previewing content during editing
- **Editor Page** - Editor page layout and functionality
- **Interactive Graph Editor** - Specialized editor for interactive graphs
- **TeX Error View** - For handling and displaying TeX errors

These editors are used by content creators to develop the interactive educational content that powers Khan Academy's learning platform.
