# Perseus Theming

This section covers theming options and customization available for Perseus components.

Perseus components can be styled and themed to match different visual designs and accessibility requirements.

## Theme Options

Perseus supports various theming options that can be applied to customize the appearance of widgets and components. These include color schemes, typography settings, and spacing configurations.

## Applying Themes

Themes can be applied globally to ensure a consistent appearance across all Perseus components, or individually to customize specific widgets.
