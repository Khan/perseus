# RadioNew Widget

The RadioNew widget is a redesigned version of the Radio widget that allows users to select one or multiple options from a list of choices.

## Features

- Single selection mode (radio buttons)
- Multiple selection mode (checkboxes)
- Support for images in choices
- Support for math content using KaTeX
- Accessible keyboard navigation
- Responsive design for various screen sizes

## Usage

The RadioNew widget is typically used for multiple-choice questions where users need to select one or more correct answers from a list of options.
