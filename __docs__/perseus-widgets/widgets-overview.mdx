# Perseus Widgets

This section contains the interactive widgets that form the core of Perseus's assessment capabilities. Widgets allow users to interact with and answer questions in various formats.

## Available Widgets

Perseus includes many interactive widgets, such as:

- **RadioNew** - Single and multiple selection questions
- **Expression** - Mathematical expression input
- **Number Line** - Interactive number line for math problems
- **Grapher** - Function and point plotting
- **Categorizer** - Item categorization challenges
- **Matrix** - Matrix input for linear algebra
- **And many more...**

Each widget in this section includes comprehensive documentation covering:
- Overview and general information
- Accessibility considerations
- Interactive demos
- Editor interfaces
- Visual state galleries
- Internal component structure
- Regression testing

## Widget Documentation Structure

For each widget, the documentation follows a consistent structure to help developers understand, use, and extend the functionality:

1. **Overview** - Introduction to the widget, its purpose, and use cases
2. **A11Y** - Accessibility considerations and implementation details
3. **Widget Demo** - Interactive examples showing the widget in action
4. **Editor Demo** - Interface for configuring the widget in the Perseus editor
5. **Widget States Gallery** - Visual representation of the widget's different states
6. **Widget Internal Components** - Breakdown of the widget's internal architecture
7. **Regression Test** - Test cases for ensuring functionality works as expected
