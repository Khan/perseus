    • InlineIcon(__namedParameters: InlineIconProps): ReactElement

      Takes an SVG icon and renders it inline.

      You can refer to icon-paths.ts for all available SVG icons.

      We assume that the viewBox is cropped and aligned to (0, 0), but icons can be defined differently. At some point we might want to add these attributes to icon-paths.js, but for now this is a fairly safe assumption.
